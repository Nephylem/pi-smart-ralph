## Learnings
- Smart Ralph TUI surfaces are concentrated in `extensions/ralph-specum/index.ts`: `notify`, status animation, footer, native task widget, and custom `ralph-subagents` widget.
- `/ralph-*` workflow commands run through `startRalphCoordinatorJob`, which starts UI surfaces then detaches workflow execution so the main Pi session can continue.
- pi-subagents supports event-driven widgets, `widgetMode: "background"`, FleetView, RPC spawn, and `Symbol.for("pi-subagents:manager")`; Ralph already relies on these integration points.
- Verification should use mocked Pi UI/event bus tests plus `npm run typecheck`, `npm test`, `npm run verify:index`, `npm run verify:pack`, and publish bundle verifier.
